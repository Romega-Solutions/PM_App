# 🌸 PinayMate - Dating App

> React Native dating app connecting Filipino women with foreign men

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Run on devices
npm run ios
npm run android
```

---

## 🔧 Environment Setup

Create `.env` file:

```env
EXPO_PUBLIC_SUPABASE_URL=https://<project-ref>.supabase.co
EXPO_PUBLIC_SUPABASE_ANON_KEY=<anon-key>
```

Use only the public anon key. `.env` is already git-ignored.

## 🏗️ Tech Stack

- **Framework:** React Native + Expo SDK 54+
- **Router:** expo-router (file-based)
- **Styling:** React Native `StyleSheet` + centralized theme tokens
- **Backend:** Supabase (PostgreSQL + Auth + Storage)
- **State:** Zustand

## 🔐 Auth Flow

1. **User Type Selection** → Filipina or Foreign Man
2. **Sign Up** → Email + Password
3. **Email Verification** → Verify via email link
4. **Account Setup** → Complete profile (photos, location, preferences)
5. **Main App** → Discover, match, chat

**Features:**

- Email verification with deep linking
- Auto sign-out before signup to prevent conflicts
- Session persistence with AsyncStorage

---

## 🗄️ Database Tables

```sql
profiles          # User profiles (extends auth.users)
match_preferences # Match filters and relationship preferences
profile_photos    # Profile photos
swipes            # Like/pass actions
matches           # Mutual likes
conversations     # Chat threads
messages          # Chat messages
verifications     # ID/selfie verification status
```

**Auto-match trigger:** Creates match when both users like each other
